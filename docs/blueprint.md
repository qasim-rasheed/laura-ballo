# **App Name**: Laura Ballo

## Core Features:

- Landing Page Content Management: Sanity.io CMS integration for managing landing page text, images, and other content.
- Bilingual Support: Seamless English/French toggle for all landing page content.
- French Lessons Booking Integration: Calendly integration to book french lessons
- Newsletter Signup Form: Custom React form that directly feeds to ConvertKit for newsletter subscriptions.
- Podcast Feed: Automated fetching of podcast RSS feed from Spotify to display latest episodes.
- AI Leadership Coach Chatbot: AI chatbot powered by OpenAI API to answer visitor questions about leadership.

## Style Guidelines:

- Primary color: Deep violet (#673AB7), signifying leadership and vision.
- Background color: Off-white (#F5F5F5), to maintain a clean and professional aesthetic.
- Accent color: Soft red (#F44336), used to draw attention to key actions and links.
- Headline font: 'Playfair' serif for a sophisticated, high-end feel.
- Body font: 'PT Sans' sans-serif as supporting text for Playfair
- Glassmorphism effects for cards and panels, adding depth and interactivity. Use shadows and blurs.
- Interactive elements that respond to user mouse movements, increasing engagement. subtle and professional only
- Smooth transitions when navigating between page sections.
